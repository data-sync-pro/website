Data Model

Data Sync Pro uses custom objects to store the metadata (Connection, Direction, Job, Executable, Field Mapping, Schedule, Job Schedule, Executable Schedule, Apex Schedule) which is used to run the app, and to log the details(Job Execution, Execution, Batch Execution) during execution. This section explains the data models in the context of different usages at a high level.

  • Directional Data Processing
  • Single Connection Actions
  • Schedule Job Management
