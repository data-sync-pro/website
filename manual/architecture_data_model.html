Data Model

Data Sync Pro uses custom objects to store the metadata (Connection, Direction, Pipeline, Executable, Field Mapping, Schedule, Pipeline Schedule, Executable Schedule, Apex Schedule) which is used to run the app, and to log the details(Pipeline Execution, Execution, Batch Execution) during execution.
