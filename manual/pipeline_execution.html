Execution

A Pipeline Execution(pushtopics__PipelineExecution__c) is a logging object that tracks the status of execution for a Pipeline, which consists of a list of sequentially ordered Mappings. Each Mapping has a corresponding Execution record created during the Pipeline's execution to track the individual Mapping's execution status. And all of those Mapping Execution records are assocated with the Pipeline Execution.
