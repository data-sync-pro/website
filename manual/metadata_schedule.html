Schedule

A Schedule(pushtopics__Schedule__c) record defines the time and frequency and when it's associated with a Pipeline or Executable, the Pipeline or Executable will be scheduled to run at the times defined in the Schedule record. If there is any update to the Schedule causing changes to the scheduled time and frequency, DSP will remove the related Pipeline Schedules and Executable Schedules, create new replicas to delete the old scheduled Apex Jobs and create new ones.
