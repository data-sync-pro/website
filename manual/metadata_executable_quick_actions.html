The following are the quick actions available on the Mapping record page.

Execute

Opens the Status Tracker modal and starts execution based on the object level settings on the Mapping record and its associated Field Mappings.

If the Mapping is batchable, the execution runs in batch mode, where the Execution record is created in the back and the Status Tracker keeps the latest results (and corresponding hyperlink) at the top. The source data is split into batches based on the Batch Size configured on the Mapping. Whenever a batch completes its execution, the Status Tracker updates by adding the Batch Execution result right below the Execution result and update the summary of the Execution result. The maximum number of batch results shown in the Status Tracker is 200, after the first 200 batches, when a new batch result comes in, the earliest successful batch result is removed from the tracker. To view the full list of batches, go to the Related List on Execution record page.

If the Mapping is not batchable, the execution runs in non-batch mode, where instead of tracking the results of both Execution and Batch Execution, only the Execution result is tracked.

Click the hyperlink on the Execution Name to see the result of the Execution; And click the hyperlink on the Batch Name to see the details of the Batch Execution.

Close

Click the Close button to close the Status Tracker modal, but note that after the modal is closed, the execution continues in the back.

Stop Execution

Click the Stop Execution button to stop the execution which is running in batch mode.

Note:

If you wish to stop the Execution, you may either click the Stop Execution button, or check the Stopped field and save the execution record if it is yet completed, or call the stopExecution method on the pushtopics.ExecutionService class using Apex.

Validate

Validates the settings of Job and its related Executables and their Field Mappings. Errors will be toasted if the validation fails.

Check Source Count

Opens the modal that shows the source record count based on the Source Object Name and Retrieve Parameters defined on the Mapping.

Note:

Even if the Retrieve Limit is set to a static value, the Check Source Count still shows the total count of the source records based on the Retrieve Parameters without considering the Retrieve Limit.

Clone Mapping

Opens the modal to fill in a few basic fields based on the current Mapping's settings and clone the Mapping along with all its associated Field Mappings upon clicking the "Clone Mapping" button.

Delete Executions

Delete all the Executions and their Batch Executions associated with the Mapping.

Edit

The standard Salesforce Edit quick action.

Delete

The standard Salesforce Delete quick action. As a result, it also cascade deletes all the related records with Master-Detail relationships.
