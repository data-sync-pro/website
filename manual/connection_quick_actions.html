Quick Actions

There is one custom quick action, Test Connection, and three standard quick actions on the Connection page layout.

Test Connection

Click the Test Connection quick action to check whether the auth setting on the Connection record is correct or not. If settings are correct, a success message will be shown; otherwise, an error message will be displayed. Read the error message to triage the issues in case of a failure.

Edit

Salesforce standard Edit quick action.

Clone

Salesforce standard Clone quick action.

Delete

Salesforce standard Delete quick action.
