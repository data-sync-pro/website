Quick Actions

Execute

Opens the Status Tracker modal and starts executing the associated Mappings in sequantial order based on the Seq No. values. As long as there is a result from a Mapping's execution, the Tracker updates with the latest status. Click the hyperlink of any Execution to check the details.

In the back end, DSP creates a Job Execution record, along with the Execution records for the associated Mappings.

Close

Click Close button to close the Status Tracker modal, but note that after the modal is closed, the Job Execution still continues in the back.

Stop Execution

Click the Stop Execution button to stop the remaining Executions..

Note:

If you wish to stop the remaining Executions, you may either click the Stop Execution button, or check the Stopped field and save the Job Execution record if it is not completed, or call the stopJobExecution method on the pushtopics.ExecutionService class using Apex.

Validate

Validates the settings of Job and its related Executables and their Field Mappings. Errors will be toasted if the validation fails.

Check Source Counts

Opens the modal that shows the source record counts based on the Retrieve Parameters configured on the Mappings associated with the Job.

Note:

For each Mapping, even if the Retrieve Limit is set to a static value, the count value is still the total count of the source records based on the Retrieve Parameters without considering the Retrieve Limit.

Clone Job

Opens the modal to fill in a few basic fields based on the current Job's settings and clone the Job along with all its associated Mappings and their Field Mappings upon clicking the "Clone Job" button.

Delete Executions

Delete all the associated Job Executions and their Executions, Batch Executions.

Delete Mappings

Delete all the associated Mappings and their Field Mappings, Executions, Batch Executions.

Edit

The standard Salesforce Edit quick action.

Delete

The standard Salesforce Delete quick action. As a result, it also cascade deletes all the related records with Master-Detail relationships.
