Is Current Org

The "Is Current Org?" checkbox is a straightforward field on the Connection object. Once checked, DSP operates without requiring credentials. It conducts queries and DML operations based on the currently logged-in user's context, indicating that the connection pertains to the current org where Data Sync Pro is installed. All security settings applicable to that user are enforced during executions.
