Pipeline

A Pipeline (designated as pushtopics__Pipeline__c) is a collection of Executables that run in a sequence. This order of execution guarantees that parent data migrates before child data, thus preserving referential integrity (relationships).

The system provides the option to halt execution upon encountering a failed Executable. Following the completion of execution, it can be configured to send out email notifications. There are multiple avenues for Pipeline execution: it can be manually operated from the user interface, automatically triggered by adding to Schedule records for preset timings, or invoked via Apex classes.

The Pipeline Builder, a utility tool, aids in the rapid and efficient creation of a list of Executables. Selections made from the objects' tree view determine the automatic population of relational fields, simplifying the process of creating Pipeline Executables. Additional details regarding this process can be found in the reference materials for the Pipeline Builder.
