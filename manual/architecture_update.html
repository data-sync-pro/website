Update

An Update Executable is designed for modifying data that already exists within the target object. It necessitates the definition of a Target Matching Field and its mapping, which Data Sync Pro utilizes to verify the presence of the record in the target object. When a record is identified in the target object, DSP proceeds with the update, applying changes according to the specified field-level mappings. Conversely, if the record does not exist in the target object—indicating it is a new entry—DSP will take no action, thereby ensuring that only existing records are updated. This functionality underscores Data Sync Pro's capability to efficiently manage and update data within targeted objects while maintaining data integrity.
