Username & Password Flow

Data Sync enables the use of the (username) and (password + security token) flow, leveraging Salesforce's secure feature - Named Credential. Follow the steps outlined below to configure:

  • Establish a Named Credential:
    • URL: Use "https://test.salesforce.com" for a sandbox environment or "https://login.salesforce.com" for production.
    • Identity Type: Select "Named Principal".
    • Authentication Protocol: Choose "Password Authentication".
    • Username: Enter the main username of the connected organization.
    • Password: Input the user's password combined with the security token. If a security token is not applicable, omit it.
  • Create a DSP Connection Record: Name it after the Named Credential you've just set up.
  • Test Connection: Verify that the setup works as expected.
